**TeachInHome（家里教）** 是一款基于微信小程序生态构建的家教辅导类对接平台。该项目采用**前端原生小程序架构 + 微信云开发**的 Serverless 模式，集成了家教订单发布与接单、社区论坛交流、个人简历管理以及管理员后台数据治理等模块。

运行这个项目需要以下步骤：

1. 环境准备
安装工具：下载并安装最新版的 微信开发者工具。
获取凭证：准备一个正式的微信小程序 AppID（注：云开发功能不支持测试号）。

2. 项目导入与基础配置
导入项目：在微信开发者工具中选择“导入项目”，选中 TeachInHome 根目录，填入您的 AppID。
开通云开发：点击开发者工具顶部的“云开发”按钮，开通服务并创建一个新的云环境，记录下云环境 ID。
修改环境变量：打开 miniprogram/app.ts 文件，找到云开发初始化逻辑，将 env 的值替换为您刚刚获取的真实云环境 ID。

3. 云端资源部署（核心步骤）
由于项目采用 Serverless 架构，必须先将后端逻辑部署到微信服务器：
部署云函数：
在开发者工具的资源管理器中，右键点击 cloudfunctions 文件夹。
选择 “同步云函数列表”。
针对里面的每一个云函数目录（如 adminLogin、publishOrder、initAdminData 等），右键选择 “云函数增量上传并安装依赖”。
初始化数据库：
打开云开发控制台的“数据库”面板。
严格按照项目文档 docs/database-index-recommendations.md 的说明，创建对应的集合（Collections）并配置数据库索引，以保障复杂查询（如订单匹配、发帖列表）的性能。

4. 前端依赖构建
本项目使用了 TDesign UI 组件库。导入后，必须点击开发者工具顶部菜单栏的 “工具” -> “构建 npm”。
等待构建生成 miniprogram_npm 目录后，前端页面才能正确解析和渲染 UI 组件。

5. 系统初始化与运行
编译预览：完成上述步骤后，点击工具栏的 “编译”，即可在左侧模拟器中看到小程序大厅界面。
创建管理员：作为平台方，首次使用需要初始化管理员权限。您可以根据 docs/创建管理员账号.md 和 docs/管理员登录操作指南.md 的指引，通过执行 initAdminData 云函数来生成超级管理员账号，随后即可通过 pages/admin 专属入口登录后台进行数据治理。
