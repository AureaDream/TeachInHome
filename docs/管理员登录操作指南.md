本文档详细说明了家教接单平台管理员登录功能的使用方法和相关配置。管理员登录功能基于微信小程序云开发实现，支持用户名密码验证和权限管理。

- **权限**: 用户管理、订单管理、论坛管理、系统管理

## 管理员登录操作步骤

### 1. 进入登录页面

1. 打开家教接单平台小程序
2. 在登录页面底部找到「管理员入口」按钮
3. 点击「管理员入口」按钮

### 2. 输入登录信息

在弹出的管理员登录对话框中：

1. **用户名**: 输入管理员用户名（默认：`admin`）
2. **密码**: 输入管理员密码（默认：`admin123`）
3. 点击「登录」按钮

### 3. 登录成功

登录成功后：
- 系统显示「登录成功」提示
- 自动跳转到管理员后台页面
- 管理员信息保存到本地存储
- 更新最后登录时间

## 管理员权限说明

管理员登录后具有以下权限：

- **用户管理** (`user_manage`): 查看和管理用户信息
- **订单管理** (`order_manage`): 发布、编辑、删除订单
- **论坛管理** (`forum_manage`): 管理论坛帖子和评论
- **系统管理** (`system_manage`): 系统配置和数据管理

## 数据库结构

### admins 集合

管理员信息存储在 `admins` 集合中，包含以下字段：

```javascript
{
  "_id": "管理员ID",
  "username": "用户名",
  "password": "密码（明文，生产环境应加密）",
  "nickName": "显示名称",
  "avatarUrl": "头像URL",
  "email": "邮箱",
  "phone": "手机号",
  "permissions": ["权限列表"],
  "status": "账号状态（active/inactive）",
  "createTime": "创建时间",
  "lastLoginTime": "最后登录时间",
  "createdBy": "创建者"
}
```

### users 集合

管理员登录后，同时在 `users` 集合中创建或更新用户记录，用于统一用户管理：

```javascript
{
  "_id": "用户ID",
  "nickName": "管理员昵称",
  "avatarUrl": "头像URL",
  "loginType": "admin",
  "role": "admin",
  "isAdmin": true,
  "_openid": "微信OpenID",
  "createTime": "创建时间",
  "lastLoginTime": "最后登录时间"
}
```

## 安全注意事项

1. **密码安全**: 生产环境中应使用加密密码存储
2. **权限控制**: 严格控制管理员权限，定期审查
3. **登录日志**: 记录管理员登录日志，便于审计
4. **账号管理**: 定期更换管理员密码，禁用不活跃账号

## 故障排除

### 常见问题

1. **登录失败 - 用户名或密码错误**
   - 检查用户名和密码是否正确
   - 确认管理员账号状态为 `active`

2. **登录失败 - 管理员账号不存在**
   - 执行 `initAdminData` 云函数初始化管理员数据
   - 检查 `admins` 集合是否存在数据

3. **云函数调用失败**
   - 检查云开发环境ID是否正确
   - 确认云函数已正确部署
   - 查看云函数日志排查错误

### 调试方法

1. **查看云函数日志**
   - 在微信开发者工具的云开发控制台中查看函数日志
   - 检查错误信息和调用参数

2. **数据库查询**
   - 在云开发控制台的数据库中查看 `admins` 集合
   - 确认管理员数据是否正确创建
