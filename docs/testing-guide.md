# 个人信息编辑功能测试指南

## 环境准备

### 1. 微信开发者工具
- 下载并安装微信开发者工具
- 使用微信扫码登录开发者工具
- 导入项目：选择 `g:\WechatProject\3` 目录

### 2. 云开发环境
- 确保云开发环境ID：`cloud1-3gqdqvkpbeab224c` 已正确配置
- 在微信开发者工具中开启云开发功能
- 确认数据库权限设置正确

## 测试步骤

### 1. 启动项目
1. 打开微信开发者工具
2. 导入项目目录：`g:\WechatProject\3`
3. 选择小程序项目类型
4. 点击「编译」按钮启动项目

### 2. 登录测试
1. 首次进入会跳转到登录页面
2. 点击「微信一键登录」按钮
3. 确认登录成功后跳转到个人中心页面

### 3. 编辑个人信息功能测试

#### 3.1 正常编辑流程
1. 在个人中心页面点击「编辑个人信息」按钮
2. 验证弹窗正常显示，表单数据正确预填充
3. 修改各个字段：
   - 昵称：输入新的昵称
   - 手机号：输入有效的手机号
   - 邮箱：输入有效的邮箱地址
   - 个人简介：输入个人简介内容
4. 点击「确认」按钮
5. 验证更新成功提示
6. 确认页面数据已更新

#### 3.2 表单验证测试

**昵称验证：**
- 留空昵称，点击确认 → 应提示「请输入昵称」
- 输入单个字符 → 应提示「昵称至少需要2个字符」
- 输入超过20个字符 → 应提示「昵称不能超过20个字符」

**手机号验证：**
- 输入无效格式（如：123456789） → 应提示「请输入正确的手机号格式」
- 输入有效格式（如：13812345678） → 应验证通过
- 留空手机号 → 应允许保存

**邮箱验证：**
- 输入无效格式（如：test@） → 应提示「请输入正确的邮箱格式」
- 输入有效格式（如：test@example.com） → 应验证通过
- 留空邮箱 → 应允许保存

**个人简介验证：**
- 输入超过200个字符 → 应提示「个人简介不能超过200个字符」
- 输入正常长度内容 → 应验证通过
- 留空个人简介 → 应允许保存

#### 3.3 取消编辑测试
1. 点击「编辑个人信息」按钮
2. 修改部分字段内容
3. 点击「取消」按钮
4. 验证弹窗关闭，数据未保存
5. 再次打开编辑弹窗，确认显示原始数据

#### 3.4 登录状态测试
1. 清除本地存储中的用户信息
2. 刷新页面或重新进入个人中心
3. 点击「编辑个人信息」按钮
4. 验证提示「请先登录」并跳转到登录页面

### 4. 错误处理测试

#### 4.1 网络异常测试
1. 断开网络连接
2. 尝试保存个人信息
3. 验证显示网络错误提示

#### 4.2 权限异常测试
1. 修改云数据库权限设置（如果可能）
2. 尝试保存个人信息
3. 验证显示权限错误提示

## 常见问题

### 1. 云开发环境连接失败
- 检查环境ID是否正确：`cloud1-3gqdqvkpbeab224c`
- 确认微信开发者工具已开启云开发功能
- 检查网络连接是否正常

### 2. 数据库权限问题
- 确认云数据库中 `users` 集合的读写权限
- 检查用户是否有更新自己信息的权限

### 3. 表单数据不显示
- 检查用户信息是否正确加载
- 确认 `userInfo._id` 字段存在
- 查看控制台是否有错误日志

### 4. 更新失败
- 检查网络连接
- 确认云开发环境状态
- 查看详细错误信息
